# Proposal: 暗黑地牢卡牌爬塔

## Why

需要一个无需安装、打开浏览器即玩的 Roguelike 卡牌爬塔游戏。暗黑奇幻主题 + 三职业差异化 + 分支路线选择 + 完善的构筑深度，目标是做出一个能让人"再来一局"的网页端《杀戮尖塔》式体验。纯前端实现，存档用 localStorage，架构预留扩展空间，后续可持续扩展新职业、新卡牌、新事件及联机模式。

## What Changes

从零构建完整的暗黑地牢卡牌爬塔网页游戏，涵盖：
- 主菜单（选模式/难度/职业）+ 职业风味描述
- 分支路线地图系统（3 章：墓穴→深渊→王座）
- 回合制能量战斗（显示敌人意图、护甲系统、中毒/灼烧/状态效果）
- 三职业（战士/法师/盗贼），各 16 张专属卡牌 + 5 种诅咒牌
- 8 种节点类型（战斗/精英/BOSS/宝箱/商店/休息/事件/诅咒事件）
- BOSS 章节制（每章 2-3 个 BOSS 随机抽取）
- 遗物系统（23 个）+ 药水系统（9 种）
- 12 个随机事件 + 商店以血换物
- 分数系统 + 详细战绩统计 + 局外解锁
- localStorage 自动存档

## Scope

### 当前版本

| 做 | 不做（当前版本） |
|---|------|
| 纯前端单页面应用 | 后端/多人/联网（后续版本扩展） |
| 3 职业 × 3 模式 × 3 难度 | 更多模式/职业（后续扩展） |
| 59 张卡牌 + 27 个遗物（含解锁） | 超过此数量的内容 |
| localStorage 存档 | 云端同步（后续扩展） |
| 章节制爬塔 | 无尽模式（后续扩展） |
| 简单/普通：敌人数值缩放 | — |
| 困难：数值缩放 + 随机 AI 变化/新招式 | — |

## Capabilities

- `combat-system`：回合制能量战斗核心
- `card-system`：卡牌池、牌组构筑、稀有度、升级、消耗
- `map-system`：分支路线地图、章节切换
- `enemy-system`：敌人 AI、意图循环、数值缩放、困难模式随机变化
- `relic-potion-system`：遗物被动效果、药水主动使用
- `event-system`：随机事件、商店、休息点
- `meta-system`：模式/难度选择、分数、战绩、存档、局外解锁
- `ui-system`：暗黑奇幻视觉、战斗 UI、地图 UI、菜单 UI

## Impact

全新项目，无遗留代码。技术栈：HTML + CSS + JavaScript，无框架依赖。设计文档（design.md 外部）已完成，覆盖所有 gameplay 维度。

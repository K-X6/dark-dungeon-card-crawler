# 暗黑地牢 — 卡牌爬塔

> 一款暗黑奇幻主题的 Roguelike 卡牌爬塔网页游戏。受《杀戮尖塔》启发，纯前端实现，浏览器打开即玩。

## 快速开始

直接在浏览器中打开 `index.html` 即可开始游戏。无需安装、无需服务器、无需构建工具。

## 游戏玩法

选择职业，在分支路线的地牢中逐层推进。通过战斗获取卡牌、遗物和药水来构筑你的牌组。每场战斗中，敌人的下回合意图完全可见——根据情报决定攻防是制胜关键。

### 三职业

| 职业 | HP | 能量 | 风格 |
|------|-----|------|------|
| 战士 | 80 | 3 | 近战直伤、护甲堆叠、力量增益 |
| 法师 | 60 | 4 | 法术 AOE、元素伤害、过牌引擎 |
| 盗贼 | 70 | 2 | 中毒叠加、敏捷连击、闪避反击 |

### 三模式 × 三难度

| 模式 | 层数 | BOSS |
|------|------|------|
| 迷你 | 5-7 层 | 1-2 个 |
| 短局 | 8-12 层 | 3-4 个 |
| 标准 | 15-20 层 | 5-7 个 |

每种模式可选简单/普通/困难难度。困难模式下敌人会获得随机 AI 变化。

### 核心机制

- **能量制**：每回合 2-4 点能量（职业不同），打牌消耗
- **护甲**：每回合清零，可叠加无上限
- **敌人意图可见**：可以看到敌人下回合要做什么
- **中毒/灼烧**：持续伤害，机制各有不同
- **遗物**：被动全局生效的道具，无持有上限
- **药水**：战斗中一次性使用，最多 3 瓶
- **商店以血换物**：没有金币，用 HP 做交易
- **章节制**：墓穴 → 深渊 → 王座之间，每章独立敌池和 BOSS

### 节点类型

战斗、精英怪、宝箱房、商店、休息点（回血或升级牌）、随机事件、诅咒事件、BOSS

### 存档

进入新楼层自动保存到浏览器 localStorage。关闭后重新打开可续玩。

## 内容规模

- **59 张卡牌**（含 5 张诅咒牌 + 6 张解锁牌）
- **27 个遗物**（含职业专属和终极遗物）
- **9 种药水**
- **20 个敌人**（9 普通 + 3 精英 + 8 BOSS）
- **12 个随机事件**

## 技术栈

- 纯 HTML + CSS + JavaScript，零框架零依赖
- 事件驱动架构：引擎层与 UI 层解耦
- 效果 DSL：卡牌/遗物/事件效果统一声明式描述
- TDD 开发：80 个自动化测试覆盖引擎层
- 代码 21 文件，约 2000 行

## 项目结构

```
index.html          — 入口
css/
  main.css          — 全局样式 + 菜单 + CSS 变量（三章主题）
  battle.css        — 战斗界面样式
  map.css           — 地图界面样式
js/
  data/             — 纯数据文件（卡牌/敌人/遗物/药水/事件）
  engine/           — 核心引擎（事件总线/战斗/牌组/地图/遗物钩子）
  ui/               — 界面渲染（菜单/战斗/地图/事件/章节/战绩）
test/               — 自动化测试
changes/            — spec-superflow 规划制品（proposal/specs/design/tasks/contract）
```

# 暗黑地牢 — 卡牌爬塔

> 暗黑奇幻 Roguelike 卡牌爬塔网页游戏。受《杀戮尖塔》启发，纯前端实现，浏览器打开即玩。

## 快速开始

双击 `index.html` 在浏览器中打开。无需安装、无需服务器、无需构建工具。

## 游戏特色

### 核心玩法
- **三职业**：战士(85HP/3⚡) / 法师(65HP/4⚡) / 盗贼(70HP/2⚡)，各16张专属卡牌
- **三模式 × 三难度**：迷你(5-7层) / 短局(8-12层) / 标准(15-20层)
- **章节制**：墓穴 → 深渊 → 王座之间，每章独立敌池和视觉
- **多敌人战斗**：普通60%概率、精英50%概率出2个敌人
- **分支路线地图**：每层可选不同节点，策略路线规划

### 战斗机制
- 能量制 + 护甲系统 + 敌人意图可见
- 中毒(不减层) / 灼烧(每回合-2层) / 虚弱(-50%伤害) / 脆弱(护甲减半) / 眩晕
- 14种敌人AI意图：攻击/防御/强化/吸血/施毒/AOE/召唤/蓄力/必杀技等
- 困难模式每战随机AI变化

### 内容规模
- **62张卡牌**（含5诅咒牌 + 6张解锁牌）
- **30个遗物**（含3职业专属 + 命运之轮）
- **9种药水**
- **20个敌人**（9普通 + 3精英 + 8 BOSS）
- **15个随机事件**

### 系统功能
- 🎨 **卡牌稀有度**：白/蓝/金边框 + 攻击红底/防御蓝底/能力金底
- 🎬 **战斗动画**：卡片飞行弹道 + 粒子爆散 + 伤害飘字 + 震屏
- 🔊 **音效**：出牌/受伤/击杀/胜利 Web Audio 音效
- ⌨️ **键盘快捷键**：空格=结束回合、1-3=选牌、ESC=关闭弹窗
- 💾 **自动存档**：进新楼层自动保存，支持续玩
- 🔓 **局外解锁**：各职业通关解锁新卡牌和专属遗物
- 📊 **数据展示**：最高分、死亡次数、连胜、回合计数、牌库计数
- 🏃 **逃跑系统**：战斗中消耗20%HP逃跑
- 🛒 **商店以血换物**：6档交易，买不起闪红提示
- 🏕️ **休息点篝火**：回30%血或升级一张牌
- ⚙ **设置面板**：查看已解锁内容 + 重置进度
- 📖 **牌组查看器**：战斗中查看抽牌堆/弃牌堆/手牌

### 入场特效
- 精英/BOSS 全屏弹窗入场
- BOSS 战暗红脉动背景
- BOSS 击败金红烟花
- 玩家受击微震
- 最后击杀震屏
- 战死血色边缘渐暗

## 技术栈

- 纯 HTML + CSS + JavaScript，零框架零依赖
- 事件驱动架构：引擎层与 UI 层解耦
- 效果 DSL：卡牌/遗物/事件效果统一声明式描述
- TDD 开发：64 个自动化测试覆盖引擎层
- 24 个源文件，81 次提交

## 项目结构

```
index.html          — 入口
css/main.css        — 全局样式 + 菜单 + CSS 变量
css/battle.css      — 战斗界面 + 动画
css/map.css         — 地图界面
js/data/            — 纯数据文件（卡牌/敌人/遗物/药水/事件）
js/engine/          — 核心引擎（事件总线/战斗/牌组/地图/遗物钩子）
js/ui/              — 界面渲染（菜单/战斗/地图/事件/章节/战绩）
test/               — 自动化测试
```
